# IDC_21days-SQLChallenge
### 21-day Challenge for SQL by IDC [Indian Data Club] sponsored by DPDzero
In this Repo, I have uploaded day-wise challenges and their solution by my approach. <br>If any other approach is possible, I will load that too for diverse learning and logic building.<br> The main aim is to improve SQL knowledge by doing problems and improving our approach.
#### About data set:
Its a hospital data set. There are total 4 tables.
<br> 1. hospital_staff.csv – List of hospital staff
<br> 2. hospital_patients.csv – Patient records
<br> 3. hospital_service_weekly.csv – Weekly service-level data
<br> 4. hospital_staff_schedule.csv – Weekly schedule
